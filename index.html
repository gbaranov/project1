JS String Methods

  • .length property returns the length of a string.
  • .indexOf() returns the index of (the position of) the first occurrence of a specified text in a string.
  • .slice() extracts a part of a string and returns the extracted part in a new string.
  • .substring() is similar to slice().The difference is that substring() cannot accept negative indexes.
  • .substr() is similar to slice().The difference is that the second parameter specifies the length of the extracted part.
  • .replace() replaces a specified value with another value in a string.
  • .toUpperCase() property converts the string to upper case
  • .toLowerCase() property converts the string to lower case
  • .concat() joins two or more strings.
  • .trim() removes whitespace from both sides of a string.
  • .charAt() returns the character at a specified index (position) in a string
  • .charCodeAt() returns the unicode of the character at a specified index in a string.
  • .split() converts string to array

    • JS Array Methods

  • .toString() converts an array to a string of (comma separated) array values.
  • .join() behaves just like toString(), but in addition you can specify the separator.
  • .pop() method removes the last element from an array.
  • .push() method removes the first array element and "shifts" all other elements to a lower index.
  • .shift() is similar to slice().The difference is that the second parameter specifies the length of the extracted part.
  • .unshift() method adds a new element to an array (at the beginning), and "unshifts" older elements.
  • .splice() method can be used to add new items to an array.
  • .concat() method creates a new array by merging (concatenating) existing arrays.
  • .slice() method slices out a piece of an array into a new array.

    • JS Number Methods

  • .toString() returns a number as a string.
  • .toExponential() returns a string, with a number rounded and written using exponential notation.
  • .toFixed() returns a string, with the number written with a specified number of decimals.
  • .toPrecision() returns a string, with a number written with a specified length.
  • .valueOf() returns a number as a number.
  • .Number() returns a number, converted from its argument.
  • .parseFloat() parses its argument and returns a floating point number.
  • .parseInt() parses its argument and returns an integer.

    • JS Math Methods

  • .PI returns 3.141592653589793.
  • .round(x) returns the value of x rounded to its nearest integer.
  • .pow(x) .pow(x, y) returns the value of x to the power of y.
  • .sqrt(x) returns the square root of x.
  • .abs(x) .abs(x) returns the absolute (positive) value of x.
  • .ceil() returns the value of x rounded up to its nearest integer.
  • .floor(x) returns the value of x rounded down to its nearest integer.
  • .sin(x) returns the sine (a value between -1 and 1) of the angle x (given in radians).
  • .cos(x) returns the cosine (a value between -1 and 1) of the angle x (given in radians).
  • .min() find the lowest value in a list of arguments.
  • .max() find the highest value in a list of arguments.
  • .random() returns a random number between 0 (inclusive), and 1 (exclusive).
